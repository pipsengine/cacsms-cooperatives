# Cacsms Cooperatives

Cacsms Cooperatives is a comprehensive cooperative management system built to handle members, savings, loans, accounting, and governance.

## Getting Started

This application is built within the AI Studio environment as a Next.js (App Router) full-stack application. It leverages the power of Next.js to provide both the frontend user interfaces and the backend API logic within a unified, high-performance container.

### Modules Roadmap

1. **Auth & Setup**: Tenant management, Registration, Login, Roles & Permissions
2. **Dashboard & Member Management**: Overviews, KYC, Onboarding
3. **Core Financials**: Savings, Contributions, Payments & Receipts
4. **Loans & Welfare**: Loan processing, approvals, and welfare management
5. **Treasury & Accounting**: Bank Accounts, GL, Ledgers, Reports
6. **Governance**: Meetings, Documents, Notifications
7. **Integrations & AI**: AI Assistant, Auditing, Settings

See `ARCHITECTURE.md` for our folder structure mapping.
